# AI Image Generator using Dall-E AI

This project deal with the openai API key , Cloudinary for storing the images and uses MongoDB as an backend technology

## Tech Stacks

 * React Js
 * Tailwind css
 * MongoDB
 * OpenAi API key
 * Cloudinary
